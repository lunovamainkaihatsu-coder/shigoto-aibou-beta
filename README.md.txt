# 🤝 しごとのあいぼう β

働く人の、今日を支える小さな相棒。

## これはなに？
中小企業で働く人のための、やさしい業務サポートAIです。
仕事を管理するのではなく、**働く「あなた」を守る**ことを目的にしています。

## β版の機能
- ② 今日のタスク整理  
- ⑤ 給料逆算（今日のがんばりを見える化）  
- ⑧ 働く人へのワンポイント法令（逆引き対応）

## 使い方
1. 今日やることを入力
2. 働いた時間を入れる
3. 今日の法令ワンポイントを確認

それだけです。
少しでも、今日が楽になれば嬉しいです。

## 注意
- 本アプリはβ版です。
- 法令情報は一般的な内容であり、法的助言ではありません。
- 個別の事案は、労働局・社労士・弁護士など専門家へご相談ください。
